PRODUCTION STEPS — EVERY ORDER
1
Open the Google Sheet
Find the order row. Confirm Status = "Received."
2
Open the Production Runner
grad-production-runner.html in your browser. Fill in all fields from the sheet row.
3
Select style → Generate → Copy
Hit "Generate Storybook Prompt" or "Generate Comic Prompt." Copy the output.
4
Open the Grad Season Creator Gem
gemini.google.com → Gems → Grad Season Creator. Paste the prompt. Hit send.
5
Save the output to Google Drive
/GradSeason/Orders/[ORDER_ID]/[GRAD_NAME]-draft.gdoc
6
Run the review checklist
Check every box before approving for print.
7
Send to printer + update sheet
Books → Devon's MIL. Figurines → Tripo3d + print vendor. Update Status column.
STATUS FLOW — UPDATE SHEET COLUMN Y
Received → In Progress → Printing → Shipped

The agent watches Column Y. Every time you update it, the agent fires a progress email to the parent automatically. You never need to email them manually.

REVIEW CHECKLIST — BEFORE PRINT APPROVAL
Child's name used throughout — not "they" or "the graduate"
At least 3 real intake details appear in the story
Challenge page acknowledges real difficulty — not softened
Proud moment page hits its emotional beat
Dedication is copied word for word — not paraphrased
No generic graduation phrases anywhere in the text
Reading level matches the grade
Every illustration description is specific — not vague
Fun fact page is genuinely light and funny

GEM REVISION COMMANDS

Type these directly into the Gem chat to fix specific issues without regenerating the whole book.

"Rewrite page [N] — [describe issue]"
Rewrites a single page. Everything else stays the same.
"Page [N] is too generic — make it more specific"
Forces the Gem to pull in more real details from the intake data.
"Adjust page [N] for [grade] reading level"
Keeps the story beat, rewrites the language for the right age.
"Restore the dedication exactly: [paste text]"
Use when the Gem paraphrased the dedication instead of copying it.
"Continue from page [N] — output was cut off"
Picks up where a long output was interrupted.
"The challenge page is too soft — make it harder"
Re-engages the emotional weight of the struggle section.
